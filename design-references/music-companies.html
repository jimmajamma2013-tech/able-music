Key patterns for ABLE

Dark is premium, light is indie/technical. 15 of 20 companies default dark. The exceptions (Bandcamp, Native Instruments, TuneCore) serve specific non-consumer audiences. ABLE's dark theme is aligned with every premium streaming competitor.

One signature color = instant identity. Spotify green. SoundCloud orange. Beatport neon green. Audiomack gold. Ableton orange. ABLE needs one unmistakable accent that fans associate with the artist, not a generic platform color — which is exactly what the per-artist theming achieves.

Album art is the content, not a decoration. Spotify, Apple Music, and Beatport all make artwork the visual hero. UI chrome is explicitly subdued. ABLE's background-image mode follows the same logic — the artist's photo IS the design.

Custom type signals investment. Spotify (Circular), Beatport (Aeonik), NI (Graphik) all use licensed or bespoke typefaces. Commodity fonts like Arial or system defaults signal underfunding. ABLE's typography choice matters more than most elements.
