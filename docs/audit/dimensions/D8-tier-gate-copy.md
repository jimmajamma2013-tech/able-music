# Dimension D8 ‚Äî Tier Gate Copy
**Category:** Copy, Voice & Messaging
**Phase:** 4 (Copy)
**Status:** Draft ‚Äî code-grounded

Every locked feature in ABLE must have an upgrade overlay that names the specific thing the artist gets when they upgrade ‚Äî not a generic "Upgrade to unlock" message. The gold lock pattern (`.gold-lock-overlay`) requires: the specific value proposition in one sentence, and the plan name that unlocks it. The `GATE_COPY` object in admin.html (lines 7056‚Äì7071) defines 14 gate entries ‚Äî the copy quality varies across them. Full compliance means every locked feature has unique, specific copy, and the two static `.gold-lock-overlay` elements in the HTML (analytics heatmap at line 3426, broadcasts at line 3541) match the `GATE_COPY` system. No duplicates. No generic unlocks.
