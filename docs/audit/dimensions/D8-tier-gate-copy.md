# Dimension D8 — Tier Gate Copy
**Category:** Copy, Voice & Messaging
**Phase:** 4 (Copy)
**Status:** Not started

Every gold lock overlay on a locked feature must name the precise thing the artist gets when they upgrade — not just that something is behind a gate. "Upgrade to unlock" is a dead end. An artist reading a gold lock should finish the sentence themselves: "I would get X, which means I can do Y." Full compliance means every tier-gated panel in admin.html carries a specific value proposition, names the tier that unlocks it, links directly to the upgrade flow, and never uses generic SaaS upgrade language. The gold lock pattern must also be visually consistent, tasteful (not alarming), and never obscure the underlying feature so completely that the artist cannot understand what they are evaluating.
