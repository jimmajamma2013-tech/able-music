# Dimension G3 — CTA Deduplication Enforcement
**Category:** Core Product Logic & User Flows
**Phase:** 1 (Foundation)
**Status:** Not started

The global CTA dedupe rule states that the same URL cannot appear in more than one zone simultaneously. If a URL is added to a Hero CTA, it must be removed from Quick Action pills automatically, and vice versa. This rule applies across three zones: Hero CTAs (max 2), Quick Action pills (max 4 narrow / 6 wide), and Section Actions (max 2 per section). Enforcement must be tested in admin.html where CTAs are managed, and its effect must be verified in able-v8.html where they are displayed. Full compliance means no single URL appears twice on the fan-facing page, the admin save functions validate this rule on every write, and the artist is given an intelligible message when a dedup conflict is detected.
